# vfox-ant

A [vfox](https://github.com/version-fox/vfox) plugin for [Apache Ant](https://ant.apache.org/).

## Installation

```bash
vfox add ant
```

Or with mise:

```bash
mise use ant@latest
```

## Usage

```bash
# List available versions
vfox search ant

# Install a specific version
vfox install ant@1.10.15

# Use a version
vfox use ant@1.10.15
```

## About Apache Ant

Apache Ant is a Java library and command-line tool whose mission is to drive processes described in build files as targets and extension points dependent upon each other.
